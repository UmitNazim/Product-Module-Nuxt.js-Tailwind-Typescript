
# Knawat Product Module

## Project setup
```
npm install
```
### Compiles and hot-reloads Project for development
```
npm run serve
```
### Compiles and minifies Project for production
```
npm run build
```
### Compiles and minifies Project as Static
```
npm run generate
```
### Running unit tests
```
npm run test:unit
```
### Compiles StoryBook for development
```
npm run storybook
```
### Compiles and minifies StoryBook for production
```
npm run storybook:build
```
